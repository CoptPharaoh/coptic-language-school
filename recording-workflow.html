Two-weekend build plan

Recording + Upload Guide

This is the repeatable system for building the whole course quickly.

Weekend 1

  1. Create GitHub repo and upload this folder.
  2. Enable GitHub Pages from the repo settings.
  3. Record 3–5 sample lessons.
  4. Upload videos to YouTube as unlisted.
  5. Replace the video placeholders with YouTube embeds.
  6. Record MP3 audio for the same 3–5 lessons.
  7. Put MP3 files in assets/audio.

Weekend 2

  1. Duplicate lessons/template.html for remaining lessons.
  2. Update lesson title, Coptic text, pronunciation, and PDF links.
  3. Add audio buttons for each letter/word.
  4. Test everything on iPhone.
  5. Send the link to the priest for review.

Recommended recording setup

  • iPhone camera or screen recording
  • AirPods or lav mic
  • Canva/PowerPoint slides
  • YouTube unlisted videos
  • MP3 audio exported from Voice Memos/GarageBand

How to add a new lesson

  1. Copy lessons/template.html.
  2. Rename it, for example: lessons/ghamma.html.
  3. Change the title and Coptic text.
  4. Add the YouTube embed.
  5. Add MP3 files to assets/audio.
  6. Add a link to the lesson inside the correct level page.
Keep videos short: 3–6 minutes is ideal.
